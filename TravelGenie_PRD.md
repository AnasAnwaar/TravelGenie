# TravelGenie PRD

## Product

**TravelGenie** --- *Plan Less. Explore More.*

Agentic AI travel planning mobile application for Android and iOS.

## Vision

An AI travel concierge that researches flights, hotels, weather, events,
safety, attractions and travel content, then generates and maintains
personalized itineraries.

## Target Users

-   Solo travelers
-   Couples
-   Families
-   Business travelers
-   Backpackers

## Core Features

1.  Authentication

-   Splash
-   Onboarding
-   Login/Signup
-   Google/Apple login

2.  User Preferences

-   Age
-   Budget
-   Travel style
-   Interests
-   Food preferences
-   Accessibility
-   Language
-   Currency

3.  Trip Planning Inputs:

-   Destination
-   Start date
-   End date
-   Travelers
-   Budget

Outputs: - Flight comparisons (external links) - Hotel comparisons
(external links) - Daily AI itinerary - Attractions - Restaurants -
Local transport - Packing checklist

4.  Agent Architecture

-   Planner Agent (orchestrator)
-   Flight Agent
-   Hotel Agent
-   Weather Agent
-   Events Agent
-   Attractions Agent
-   Blog Agent
-   Video Agent
-   Safety Agent
-   Recommendation Agent

5.  Saved Trips

-   Create
-   Edit
-   Regenerate
-   Duplicate
-   Delete
-   Share

6.  Explore Personalized destination feed using preferences and search
    history.

7.  Safety Display travel advisories, political unrest, strikes,
    disasters and health alerts.

8.  Weather Modern widget with hourly/daily forecasts, UV, humidity,
    AQI, sunrise/sunset.

## Navigation

Bottom tabs: - Home - Explore - Plan - Trips - Profile
